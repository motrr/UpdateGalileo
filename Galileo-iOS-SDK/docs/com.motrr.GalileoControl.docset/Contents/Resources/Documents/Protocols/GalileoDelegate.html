GalileoDelegate Protocol Reference

Conforms to NSObject
Declared in GalileoDelegate.h

Overview

This protocol defines an interface for delegates of a Galileo object to respond to connection and disconnection events.

galileoDidDisconnect

Sent to the delegate when a previously connected Galileo accessory is disconnected.

- (void)galileoDidDisconnect

Discussion

This event will be triggered when the accessory is unplugged from the device or when the accessory’s battery level is depleted. Delegates should call waitForConnection on the Galileo object if they wish to be notified when Galileo reconnects.
